# 1 우리 가족 섬
### 프로젝트 목표
- 사이좋은 가족을 위한 메타버스 라이프로깅 서비스
- 가족 구성원의 위치와 일정을 공유하는 SNS
- AI기술의 도움으로 가족 간의 유대감 형성하고 가족 간의 소통이 더욱 즐거워지게 만드는 것
